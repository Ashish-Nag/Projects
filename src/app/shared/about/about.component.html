About The App

The first step for this app was to create the following modules and components:

  1. Admin module: Which contains a component where the admin can see the results of the users who have taken the test.
  2. Assessment Module: Contains a component of angular form, where the user can take the test.
  3. Auth Module: Contains the login and sign-up component, also contains the admin sign-in panel
  4. Shared Module: contains header, footer, a landing page, and the about page
  5. User Module: contains the profile component, which is populated after Signing up, and taking the test.
